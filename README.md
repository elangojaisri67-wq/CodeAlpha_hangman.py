# Hangman Game

## Description
This is a simple text-based Hangman game written in Python. 
The player has to guess a hidden word one letter at a time within limited attempts.

## Features
- Random word selection
- 6 incorrect guesses allowed
- Displays guessed letters
- Interactive console gameplay

## Technologies Used
Python

## How to Run
1. Download the file
2. Run using:
   python hangman.py
